Installation API :

- Ouvrez votre IDE, assurez vous d'être dans le dossier qui contient le fichier vagrant.

- Dans le terminal, entrez vagrant up.

- Ensuite assurez-vous de bien avoir le fichier concernant la création de la base de donnée, lancez la commande npm run initDB, si tout fonctionne bien, vous aurez le message done qui s'affichera.

- Vous pouvez ensuite entrer la commande npm run dev.

Voilà, tout est ok côté Backend.
